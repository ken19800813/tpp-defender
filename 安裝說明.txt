========================================
直播小幫手（TPPchat）安裝指南
========================================

【系統需求】
- macOS 10.13 以上
- Windows 10/11 以上
- 網際網路連線（首次啟動需下載瀏覽器引擎，約 200MB）


========================================
【Mac 安裝步驟】
========================================

1. 下載 DMG 檔案
   位置：https://github.com/ken19800813/tpp-defender/releases
   檔案名：TPPchat.dmg（約 133MB）

2. 首次打開時的安全警告（自動處理）
   • 應用會自動移除 macOS Gatekeeper 隔離屬性
   • 你無需執行任何 Terminal 指令
   • 直接雙擊開啟即可

3. 掛載 DMG
   • 雙擊 TPPchat.dmg
   • 會出現一個 Finder 視窗，內有 TPPchat.app
   • 將 TPPchat.app 拖到「應用程式」資料夾（推薦）
   或直接在 DMG 中雙擊執行

4. 首次啟動
   • 應用會自動下載 Playwright Chromium（約 200MB，需 5-15 分鐘）
   • 第一次下載後，登入狀態會被保存
   • 之後每次啟動都會立即打開（秒速）✓


========================================
【Windows 安裝步驟】
========================================

1. 下載 EXE 檔案
   位置：https://github.com/ken19800813/tpp-defender/releases
   檔案名：TPPchat.exe（約 43MB）

2. 首次執行時的安全警告（SmartScreen）
   Windows SmartScreen 會顯示：
   「Windows 已保護你的電腦 - 無法驗證此應用程式」

   【一鍵解決（只需做一次）】：
   • 點「更多資訊」按鈕
   • 點「仍然執行」按鈕
   • 即可啟動應用

   注意：這只會在第一次執行時出現。之後直接雙擊就能打開。

3. 首次啟動
   • 應用會自動下載 Playwright Chromium（約 200MB，需 5-15 分鐘）
   • 進度會顯示下載相關的詳細訊息
   • 第一次下載後，登入狀態會被保存
   • 之後每次啟動都會立即打開（秒速）✓

4. 快捷方式（可選）
   • 右鍵點擊 TPPchat.exe
   • 選「建立捷徑」
   • 可以移到桌面方便快速啟動

5. 防火牆提示（如果出現）
   • Windows Defender 防火牆可能會詢問
   • 點「允許」即可
   • 應用需要網路連線才能運作


========================================
【首次啟動注意事項】
========================================

1. 網路連線要求
   • 必須連上網際網路
   • 若系統未安裝 Chrome，會自動下載 Chromium（約 200MB）
   • 下載需要 5-15 分鐘，請耐心等候

2. 瀏覽器選擇（自動）
   • 應用使用 Playwright Chromium（輕量、自動登入狀態保存）
   • 首次下載後會自動保存登入狀態，之後無需重新登入

3. YouTube 帳號登入
   • 應用會開啟瀏覽器視窗
   • 登入你的 YouTube 帳號
   • 開啟一場直播（自己的或別人的）
   • 應用會開始監控該直播的聊天室

4. 貼上 YouTube 網址
   • 複製直播 URL
   • 用 **Ctrl+V**（Windows）或 **Cmd+V**（Mac）貼上
   • 或透過「編輯 → 貼上網址」菜單

5. 防火牆提示（Windows 可能出現）
   • 如果 Windows Defender 跳出提示，點「允許」
   • 這是正常的（應用需要網路連線）


========================================
【常見問題】
========================================

【Q: 執行後沒反應？】
A:
• 系統沒有 Chrome 的話會自動下載（可能需要 5-15 分鐘）
• 看看工作列/Dock 有沒有圖示，等候片刻
• 檢查網路連線是否正常
• 如果還是不動，試試重啟電腦

【Q: 出現「檔案已損壞」或「無法驗證」警告？】
A:
• macOS：應用會自動處理，無需手動操作
• Windows：按「更多資訊」→「仍然執行」即可

【Q: 下載卡住或超時？】
A:
• 檢查網速（YouTube 直播需要不錯的頻寬）
• 試試關掉 VPN 或代理
• 重新下載應用

【Q: 為什麼用 Chromium 而不用系統 Chrome？】
A:
• Chromium 能正常登入 Google 帳號（系統 Chrome 有遠程調試限制）
• 登入狀態會被自動保存到 ~/.tppchat/browser_profile
• 第二次啟動就無需重新登入，直接快速打開

【Q: Ctrl+V 不能貼上？】
A:
• 改用菜單：「編輯 → 貼上網址」
• 確認已複製 YouTube 直播網址到剪貼簿
• 試試重新啟動應用

【Q: 怎樣卸載？】
A:
• macOS：直接把 TPPchat.app 拖到垃圾筒
• Windows：刪除 TPPchat.exe 即可（無安裝程序，純單檔應用）

【Q: 應用要求「允許」權限？】
A:
• 需要攝影機或麥克風：直播功能需要（如果有的話）
• 需要網路：必需，用於監控 YouTube 聊天
• 按「允許」即可
