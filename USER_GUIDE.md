## 📡 直播監控（第一個頁籤）

**用途**：貼上 YouTube 直播網址，讓系統幫你盯著聊天室。

1. 複製你想監看的直播網址（網址需包含 `watch?v=`，例如 `https://www.youtube.com/watch?v=xxxxxxx`）
2. 貼到「YouTube 直播網址」輸入框
3. 點擊「🚀 啟動雷達」

   - 系統會先檢查這個頻道是不是黑名單頻道（例如綠營官方帳號、側翼媒體）。如果是，會直接拒絕啟動並顯示警告，不會開瀏覽器。
   - 若通過檢查，會另外開一個瀏覽器視窗顯示該場直播的聊天室——**這個視窗只用來讀取留言，不會有任何自動點擊或打字動作**。
4. 「🔔 實時日誌」會即時捲動顯示聊天室留言：
   - 🟢 綠字：一般留言
   - 🔴 紅字：被判定為側翼攻擊的留言，畫面右下角會同時彈出一個小視窗
5. 看到彈出小視窗時：
   - 視窗上會顯示對方留言內容，以及系統建議的反擊文字
   - 點擊「📋 複製反擊建議」（或按空白鍵）→ 文字會複製到你的剪貼簿
   - **自己**切換到 YouTube 聊天室視窗，貼上（Cmd+V）、確認內容、按 Enter 送出
6. 不想再監看了，點擊「⏹️ 停止」即可結束。直播結束或按下停止後，這一整場的聊天記錄會自動存檔，可以到「🗂️ 歷史記錄」頁籤查看。

> 為什麼要我自己貼上發送？因為這樣才不會被 YouTube 判定為機器人操作而被鎖帳號，也讓你在送出前有機會再看一眼、自己把關內容。

## 🛠️ 防禦規則設定（第二個頁籤）

**用途**：設定「聊天室出現什麼關鍵字 → 建議回覆什麼」的對應規則。

- 畫面上會列出目前所有生效的規則，前面標示：
  - `☁️ 雲端`：從 GitHub 同步下來的規則，人數多、由團隊統一維護，**本機無法刪除**（但你可以在雲端關鍵字檔案更新後自動跟著更新）
  - `🏠 自訂`：你自己新增的規則，只存在你的電腦裡
- **➕ 新增規則**：輸入觸發關鍵字（逗號分隔）與回覆草稿（每行一句），儲存時系統會自動檢查是否包含禁用詞（例如人身攻擊字眼），有的話會拒絕儲存
- **❌ 刪除規則**：只會列出你自己新增的規則供選擇刪除
- **✏️ 編輯規則**：目前作法是先刪除舊規則，再重新新增一筆更新後的內容

## 💬 反擊建議（第三個頁籤，原「挺台言論自訂」）

**用途**：管理系統會隨機抽出來用的反擊建議語句庫。

- 當聊天室留言命中防禦規則時，系統本來會用該規則自帶的回覆；但有 **20% 機率**會改用這份「反擊建議」語句庫裡的句子——**這個機制完全透明**，不管抽到哪一句，彈出視窗都會讓你看清楚實際文字內容，再由你決定要不要複製送出
- 上方文字框可以自訂你想用的語句，每行一句，按「💾 儲存反擊建議」立即套用到本機
- 下方「雲端預設反擊建議」區塊是唯讀預覽，讓你參考目前雲端同步下來的內容，無法直接編輯

## 🗂️ 歷史記錄（第四個頁籤）

**用途**：回顧過去每一場直播監看時，聊天室裡發生了什麼。

- 每次「啟動雷達」到「停止 / 直播結束」算一場記錄，系統會把該場**所有**聊天留言（不只側翼留言，一般留言也會記錄）存到本機的 `logs/` 資料夾
- 列表會顯示每場的節目名稱、日期、留言總數、被標記為側翼攻擊的則數
- 點擊「查看完整記錄」可以看到逐條留言，**紅字＝被判定為側翼攻擊的留言**，方便事後複盤
- 這些記錄只存在你自己的電腦上，**不會**上傳到 GitHub 或任何雲端

## 常見問題

**Q：為什麼「防禦規則設定」裡的關鍵字數量突然變多／變少？**
A：雲端規則（☁️ 標籤）是團隊在 GitHub 上維護的，只要那邊更新，你這邊重新啟動應用程式就會自動同步最新版本。

**Q：關鍵字/反擊建議可以自己改嗎？**
A：可以，「防禦規則設定」新增規則、「反擊建議」編輯語句庫都是本機自訂，不會影響雲端規則，也不會上傳回 GitHub。

**Q：畫面顯示「安全機制判定：本工具不支援此非授權親綠陣營頻道運作」是什麼意思？**
A：代表你輸入的直播網址所屬頻道在黑名單清單中，系統設計上不允許對這類頻道啟動監看/提醒功能。

**Q：歷史記錄會不會被別人看到？**
A：不會，`logs/` 資料夾已被排除在 Git 版本控制外，純粹存在你自己的電腦裡。

**Q：突然沒辦法連線同步雲端規則怎麼辦？**
A：沒關係，系統會自動改用上次成功同步、存在本機的快取內容繼續運作，不影響監看功能。
