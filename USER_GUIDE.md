# 直播小幫手：打擊青鳥人人有責 — 操作指引

真人監看＋一鍵反擊輔助工具。系統負責「看」與「提醒」，實際要不要送出仍由你在彈窗上親眼確認文字後點擊決定——**但點擊後系統會直接把文字打進聊天室並送出，這一步是真的公開發言，請留意**。

## 直播監控（第一個頁籤）

**用途**：貼上 YouTube 直播網址，讓系統幫你盯著聊天室，命中規則時自動送出反擊回覆。

1. 複製你想監看的直播網址（網址需包含 `watch?v=`，例如 `https://www.youtube.com/watch?v=xxxxxxx`）
2. 貼到「YouTube 直播網址」輸入框
3. 點擊「啟動雷達」
   - 系統會先檢查這個頻道是不是黑名單頻道。如果是，會直接拒絕啟動並顯示警告，不會開瀏覽器。
   - 若通過檢查，會另開一個瀏覽器視窗顯示該場直播的聊天室。**這個瀏覽器視窗會被自動打字＋送出訊息使用**，第一次使用若聊天室要求登入，請手動登入一次，之後會記住登入狀態。
4. 「即時日誌」會即時捲動顯示聊天室留言：
   - 白字：一般留言
   - 紅字：被判定為側翼攻擊的留言，畫面右下角會同時彈出一個小視窗
5. 看到彈出小視窗時：
   - 視窗上會顯示對方留言內容，以及系統建議要送出的反擊文字
   - 點擊「自動送出：...」（或按空白鍵）→ 系統會自動把這段文字打進聊天室輸入框並按下送出，同時把瀏覽器視窗帶到最前面讓你看到結果
   - 文字也會同時複製到剪貼簿，若自動送出失敗（例如找不到輸入框），可以自己手動貼上
6. 想確認彈窗機制本身正常，可以點「測試彈跳視窗」——這只會跳出一個測試用假彈窗，不會真的送出任何訊息（因為當時沒有真正在監看）。
7. 不想再監看了，點擊「停止」即可結束。直播結束或按下停止後，這一整場的聊天記錄會自動存檔，可以到「歷史記錄」頁籤查看。

> ⚠️ 因為送出是全自動的，命中規則的留言只要你點下彈窗按鈕，訊息就會立即公開發出，沒有第二次確認機會。請確保防禦規則跟反擊建議語句庫的內容你都能接受被公開發表。

## 防禦規則設定（第二個頁籤）

**用途**：設定「聊天室出現什麼關鍵字 → 建議回覆什麼」的對應規則，用表格呈現，支援分頁與每頁筆數選擇。

- 表格欄位：編號、偵測關鍵字、反擊內容、來源
  - 來源「雲端」：統一維護、自動同步，**本機無法刪除**
  - 來源「自訂」：你自己新增的規則，只存在你的電腦裡，可編輯或刪除
- 雙擊任一列可查看該規則完整（未截斷）的關鍵字與反擊內容
- **新增規則**：輸入觸發關鍵字（逗號分隔）與回覆草稿（每行一句），儲存時系統會自動檢查是否包含禁用詞，有的話會拒絕儲存
- **編輯規則**：直接修改既有自訂規則的內容，不需要先刪除
- **刪除規則**：只會列出你自己新增的規則供選擇刪除

## 反擊建議（第三個頁籤）

**用途**：管理系統會隨機抽出來用的反擊建議語句庫。

- 當聊天室留言命中防禦規則時，系統本來會用該規則自帶的回覆；但有 **20% 機率**會改用這份「反擊建議」語句庫裡的句子——不管抽到哪一句，彈出視窗都會讓你先看清楚實際文字內容，再決定要不要點擊送出
- 上方文字框可以自訂你想用的語句，每行一句，按「儲存反擊建議」立即套用到本機
- 下方「雲端預設反擊建議」區塊是唯讀預覽，標題旁的複製圖示可以一鍵把整份雲端語句庫複製到剪貼簿

## 歷史記錄（第四個頁籤）

**用途**：回顧過去每一場直播監看時，聊天室裡發生了什麼，支援分頁、刪除、內文搜尋。

- 每次「啟動雷達」到「停止 / 直播結束」算一場記錄，系統會把該場**所有**聊天留言（不只側翼留言，一般留言也會記錄）存到本機的 `logs/` 資料夾
- 列表顯示每場的節目名稱、日期、留言總數、被標記為側翼攻擊的則數，可調整每頁顯示筆數
- 點擊「查看完整記錄」可以看到逐條留言，紅字＝被判定為側翼攻擊的留言，並可用上方搜尋框搜尋留言內容或留言者，Enter 跳到下一筆符合結果
- 點擊「刪除」可以移除不需要的記錄（會二次確認）
- 這些記錄只存在你自己的電腦上，**不會**上傳到任何雲端

## 常見問題

**Q：為什麼「防禦規則設定」裡的關鍵字數量突然變多／變少？**
A：雲端規則是統一維護的，只要那邊更新，重新啟動應用程式就會自動同步最新版本。

**Q：關鍵字/反擊建議可以自己改嗎？**
A：可以，「防禦規則設定」新增規則、「反擊建議」編輯語句庫都是本機自訂，不會影響雲端規則，也不會回傳到雲端。

**Q：畫面顯示「安全機制判定：本工具不支援此非授權親綠陣營頻道運作」是什麼意思？**
A：代表你輸入的直播網址所屬頻道在黑名單清單中，系統設計上不允許對這類頻道啟動監看/提醒功能。

**Q：自動送出失敗會怎樣？**
A：系統會在即時日誌顯示失敗原因（例如找不到聊天室輸入框），文字仍會留在剪貼簿，你可以自己手動貼上送出。

**Q：歷史記錄會不會被別人看到？**
A：不會，`logs/` 資料夾已被排除在 Git 版本控制外，純粹存在你自己的電腦裡。

**Q：突然沒辦法連線同步雲端規則怎麼辦？**
A：沒關係，系統會自動改用上次成功同步、存在本機的快取內容繼續運作，不影響監看功能。
